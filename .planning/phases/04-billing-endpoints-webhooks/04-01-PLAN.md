Add a Webhook model to prisma/schema.prisma after the ApiKey model:

```prisma
model Webhook {
  id        String   @id @default(auto()) @map("_id") @db.ObjectId
  userId    String   @db.ObjectId
  url       String   // HTTPS endpoint to receive events
  events    String[] // ["generation.complete", "credits.depleted", "brand.updated"]
  secret    String   // HMAC-SHA256 signing secret (generated server-side)
  active    Boolean  @default(true)
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt

  @@index([userId])
  @@index([userId, active])
  @@map("webhooks")
}
```

**server/routes/billing.ts** — Create GET /balance endpoint:
- Use express.Router(), import authenticate + AuthRequest from middleware/auth.js
- Import prisma from db/prisma.js (same pattern as other routes)
- Query User by req.userId, select: monthlyCredits, creditsUsed, freeGenerationsUsed, subscriptionTier, creditsResetDate
- Return JSON:
```json
{
  "balance": {
    "monthlyCredits": 100,
    "creditsUsed": 42,
    "creditsRemaining": 58,
    "freeGenerationsUsed": 3,
    "subscriptionTier": "creator",
    "creditsResetDate": "2026-06-01T00:00:00Z"
  }
}
```
- creditsRemaining = monthlyCredits - creditsUsed (floor at 0)
- Return 404 if user not found

**server/routes/webhooks.ts** — Create CRUD endpoints:
- POST / — Register webhook. Body: { url: string, events: string[] }
  - Validate url starts with "https://"
  - Validate events is non-empty array, each value in ["generation.complete", "credits.depleted", "brand.updated"]
  - Generate secret: crypto.randomBytes(32).toString('hex')
  - Create via prisma.webhook.create({ data: { userId, url, events, secret } })
  - Return 201 with { id, url, events, secret, active, createdAt } — secret shown ONLY on creation
- GET / — List user's webhooks
  - prisma.webhook.findMany({ where: { userId } })
  - Return array WITHOUT secret field (map to omit it)
- DELETE /:id — Delete a webhook
  - prisma.webhook.deleteMany({ where: { id: req.params.id, userId } }) — ensures ownership
  - Return 204

All routes use authenticate middleware. Max 5 webhooks per user (count before create, return 400 if exceeded).

**server/app.ts** — Register both routes:
- Import billingRoutes from './routes/billing.js'
- Import webhookRoutes from './routes/webhooks.js'
- Add ['/billing', billingRoutes] and ['/webhooks', webhookRoutes] to the routes array (same pattern as usage)
