## Regras de kit para light-studio
- **Funciona bem:** kit escuro (navy, black, charcoal) sobre fundo branco
- **Evitar:** kit totalmente branco (cricket whites) sobre fundo branco — sujeito some
- **Solução para kit branco:** mudar fundo para `very pale grey` ou adicionar sombra no chão

## Regras de ação para ghosts
- **Melhor para ghost:** movimentos lineares (sprint, swing, golpe)
- **Ruim para ghost:** movimentos circulares (giro) — ghost fica confuso
- **Ideal:** um eixo de movimento claro (esquerda→direita, baixo→cima)
