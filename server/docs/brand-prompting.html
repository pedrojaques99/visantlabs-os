Brand-Aware AI Prompting

How to use Visant brand guidelines to generate on-brand mockups — without describing logos or typography in your prompt.

The Core Rule

Pass brandGuidelineId and let the system inject brand assets automatically.
Never describe the logo or font in the prompt text. The model will hallucinate them.
BAD "Add the SPORTS 248 logo with concentric circles bullseye icon in bold white Archivo font at the bottom..."
GOOD "Product post for Sports 248. Dark navy background, cricket bats centered, brand logo at bottom." + brandGuidelineId

What gets injected automatically

When you pass brandGuidelineId to mockup-generate or mockup-batch-generate, the server automatically:

  • Logos — fetched from the guideline's logos[] array and passed as referenceImages (up to 2, priority: primary → icon → light → dark)
  • Media kit images — style reference photos from media[] are appended as additional referenceImages (up to 2)
  • Brand context text — colors, typography, tone of voice, do's/don'ts, imagery guidelines, strategy/positioning, archetypes, target personas — prepended to the prompt
Total referenceImages = your own (if any) + logos (≤2) + media (≤2). OpenAI gpt-image-1 supports up to 16 reference images total.

API Usage

Single mockup

POST /api/mockups/generate
Authorization: Bearer visant_sk_...

{
  "promptText": "Product post. Dark navy background. Cricket bats centered. Brand logo at bottom.",
  "brandGuidelineId": "69e7c4af3f7d1be9a660a658",
  "provider": "openai",
  "model": "gpt-image-1",
  "aspectRatio": "1:1",
  "resolution": "1K"
}

Batch (parallel)

POST /api/mockups/batch-generate
{
  "prompts": [
    "Product post. Dark navy background. Cricket bats centered.",
    "Story poster. Sky blue background. BUILT FOR THE MATCH bold headline. Athlete with motion blur.",
    "Hero banner. Dark navy. Typography mask, sport photo through letterforms. Left-side headline."
  ],
  "brandGuidelineId": "69e7c4af3f7d1be9a660a658",
  "provider": "openai",
  "model": "gpt-image-1",
  "aspectRatio": "1:1"
}

MCP Tool Usage (Claude / Cursor)

// First — get the guideline id
list_brand_guidelines()  // or get by name

// Then — generate
generate_mockup({
  promptText: "Product post. Dark navy. Cricket bats centered.",
  brandGuidelineId: "69e7c4af3f7d1be9a660a658",
  provider: "openai",
  model: "gpt-image-1"
})

// Or batch
batch_generate_mockups({
  prompts: ["...", "...", "..."],
  brandGuidelineId: "69e7c4af3f7d1be9a660a658",
  provider: "openai",
  model: "gpt-image-1"
})

Prompt Writing Rules

DoDon't
Describe composition, layout, mood, sceneDescribe the logo visually (shape, colors, icon)
Mention the brand name as a label: "brand logo at bottom"Try to reproduce typography by describing font weight/style
Use color names from the brand ("deep navy", "action blue")Use generic color descriptions that contradict the brand
Reference product category: "cricket bats", "darts"Invent product details not in the brief
Describe photography style: "motion blur", "studio crisp"Ask for text overlays — these are unreliable with image models
